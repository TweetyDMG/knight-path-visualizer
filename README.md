# KnightMove — Визуализатор ходов шахматного коня

Интерактивное JavaFX-приложение для моделирования и анимации перемещений шахматного коня по доске произвольного размера (от 3×3 до 20×20). Позволяет расставлять стены, задавать глубину BFS-обхода, визуализировать все достижимые клетки с цветовой индикацией ветвей и анимировать перемещение фигуры по выбранному пути. Демонстрирует работу с графами (BFS), JavaFX (FXML, CSS, Timeline-анимации) и кастомные бордерлесс-окна.

## 🛠 Технологический стек

При проектировании архитектуры приложения упор делался на скорость обработки запросов и модульность.

*   **Язык разработки:** Java 23
*   **Фреймворки:** JavaFX 17.0.6 (Controls, FXML, CSS, Web, Media)
*   **Инструменты тестирования:** JUnit 5.10.2
*   **Сборка:** Apache Maven 3.9+ (Maven Wrapper)
*   **Доп. библиотеки:** ControlsFX 11.2.1
*   **Модульность:** Java Platform Module System (JPMS)

---

## 🚀 Ключевой функционал

*   **Управление доской:** Шахматная доска произвольного размера от 3×3 до 20×20
*   **Автоматизация логики:** BFS-обход графа — поиск всех достижимых клеток за N ходов с фиксацией глубины и индекса ветви
*   **Интеграции:** Кастомное бордерлесс-окно, цветовая индикация ветвей (8 направлений хода коня)
*   **Валидация и безопасность:** Плавные анимации на `Timeline`, `FadeTransition`, `FillTransition`

## 📁 Архитектура и структура проекта

В проекте используется компонентная архитектура с разделением на модель, контроллер и представление (FXML).

```text
KnightMove/
├── pom.xml                        # Maven-конфиг (Java 23, JavaFX 17, JUnit 5)
├── mvnw / mvnw.cmd                # Maven Wrapper
├── run.sh                         # Скрипт сборки и запуска
├── .mvn/wrapper/                  # Настройки Wrapper'а
├── README.md                      # Текущая документация
├── src/
│   ├── main/java/org/example/lab8javafx/
│   │   ├── module-info.java              # Java module descriptor (JPMS)
│   │   ├── KnightApp.java                # Точка входа / JavaFX Application
│   │   ├── KnightController.java         # Главный контроллер GUI
│   │   ├── GameBoard.java                # Модель доски + BFS-алгоритм
│   │   ├── KnightMove.java               # DTO: координаты, глубина, parent
│   │   └── MovesWindowController.java    # Контроллер окна со списком ходов
│   ├── main/resources/org/example/lab8javafx/
│   │   ├── KnightGUI.fxml                # Главная форма
│   │   ├── MovesWindow.fxml              # Форма окна ходов
│   │   ├── style.css                     # Стилизация
│   │   └── knight.png                    # Спрайт коня
│   └── test/java/org/example/lab8javafx/
│       └── GameBoardTest.java            # Модульные тесты BFS-алгоритма
└── .gitignore
```

### Логическая схема BFS-обхода

```
                    [start] (x, y, depth=0)
                         │
            ┌────────────┼────────────┐
            │ dx[]/dy[]  │  ... 7 веток│
            ▼            ▼             ▼
      (x±2, y±1)   (x±1, y±2)     ...
            │            │
            ▼            ▼
      visited[][] ── не посещать повторно
      walls[][]   ── не ходить через стены
       depth++    ── до maxDepth
```

## 💻 Локальное развертывание

Для запуска проекта в изолированном окружении вам понадобятся **Java 23+** и **Apache Maven 3.9+**.

### 1. Клонирование репозитория

```bash
git clone https://github.com/<ваш-username>/KnightMove.git
cd KnightMove
```

### 2. Сборка и запуск

```bash
# Сборка и прогон тестов
./mvnw clean verify

# Запуск
./run.sh

# Или напрямую через Maven:
./mvnw clean org.openjfx:javafx-maven-plugin:0.0.8:run
```

### 3. Сборка исполняемого JAR

```bash
./mvnw clean package
java --module-path target/modules \
     -m org.example.lab8javafx/org.example.lab8javafx.KnightApp
```

### Настройки по умолчанию

| Параметр | Значение | Диапазон |
|----------|----------|----------|
| Размер доски | 8 × 8 | 3–20 |
| Глубина обхода | 2 хода | 0–100 |
| Ветви (направления) | 8 | фиксировано (L-ходы коня) |

## 🔌 Управление приложением

| Действие | Результат |
|----------|-----------|
| **ЛКМ** по клетке | Установить / переместить коня |
| **ПКМ** по клетке | Установить / снять стену |
| **«Рассчитать»** | Подсветить все клетки, достижимые за N ходов |
| **«Показать путь»** | Анимация обхода по всем доступным веткам |
| **«Изменить размер»** | Перестроить доску с новыми размерами |
| **«Очистить поле»** | Сбросить все отметки посещений |
