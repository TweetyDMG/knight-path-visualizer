# KnightMove — Визуализатор ходов шахматного коня

Интерактивное JavaFX-приложение для моделирования и анимации перемещений шахматного коня по доске произвольного размера (от 3×3 до 20×20). Позволяет расставлять стены, задавать глубину BFS-обхода, визуализировать все достижимые клетки с цветовой индикацией ветвей и анимировать перемещение фигуры по выбранному пути.

Проект демонстрирует: работу с графами (BFS), JavaFX (FXML, CSS, Timeline-анимации), кастомные бордерлесс-окна и компонентную архитектуру десктопного GUI.

## 🚀 Ключевой функционал

- **Кастомное бордерлесс-окно** — собственные кнопки «Свернуть»/«Закрыть», перетаскивание мышью за любую область
- **Шахматная доска произвольного размера** — от 3×3 до 20×20 с плавным изменением через спиннеры (`FadeTransition`)
- **Расстановка стен** — правая кнопка мыши по клетке с анимацией затемнения; конь не может проходить сквозь стены
- **BFS-обход графа** — поиск всех достижимых клеток за N ходов с фиксацией глубины и индекса ветви
- **Цветовая индикация ветвей** — каждая из 8 ветвей (направлений хода коня) окрашивается своим базовым цветом; чем глубже ход — тем темнее оттенок (интерполяция к чёрному)
- **Анимация перемещения** — плавная `Timeline`-анимация по клику на клетку или выбору хода в списке
- **Окно со списком ходов** — модальное окно со всеми достижимыми клетками; клик по строке подсвечивает путь на доске и включает анимацию
- **Персистентные отметки** — при перемещении коня клетка запоминает цвет ветви первого посещения; очистка — одной кнопкой
- **Плавные переходы** — `FadeTransition` при ресайзе доски, `FillTransition` при установке стен

### Логическая схема BFS-обхода

```
                    [start] (x, y, depth=0)
                         │
            ┌────────────┼────────────┐
            │ dx[]/dy[]  │  ... 7 веток│
            ▼            ▼             ▼
      (x±2, y±1)   (x±1, y±2)     ...
            │            │
            ▼            ▼
      visited[][] ── не посещать повторно
      walls[][]   ── не ходить через стены
       depth++    ── до maxDepth
```

Каждый ход представлен объектом `KnightMove` со ссылкой на родительский ход (односвязный список) — это позволяет реконструировать полный путь от начальной позиции до любой клетки.

## 💻 Локальное развёртывание

### Требования

- **Java 23+** (SDK): [Oracle JDK](https://www.oracle.com/java/technologies/downloads/) / [OpenJDK](https://jdk.java.net/)
- **Apache Maven 3.9+** (или используйте встроенный `./mvnw`)

### Быстрый старт

```bash
# 1. Клонирование
git clone https://github.com/username/KnightMove.git
cd KnightMove

# 2. Сборка и прогон тестов
./mvnw clean verify

# 3. Запуск (скрипт — всегда работает, не зависит от прокси)
./run.sh

# Или напрямую через Maven:
./mvnw clean org.openjfx:javafx-maven-plugin:0.0.8:run
```

### Сборка исполняемого JAR

```bash
./mvnw clean package
# Исполняемый JAR появится в target/
java --module-path target/modules \
     -m org.example.lab8javafx/org.example.lab8javafx.KnightApp
```

### Настройки по умолчанию

Все параметры конфигурируются через GUI. Начальные значения:

| Параметр          | Значение | Диапазон         |
|-------------------|----------|------------------|
| Размер доски      | 8 × 8    | 3–20 строк/столбцов |
| Глубина обхода    | 2 хода   | 0–100            |
| Ветви (направления) | 8        | фиксировано (L-ходы коня) |

## 🎮 Управление

| Действие                     | Результат                                         |
|------------------------------|---------------------------------------------------|
| **ЛКМ** по клетке            | Установить / переместить коня                     |
| **ПКМ** по клетке            | Установить / снять стену                          |
| **«Рассчитать»**             | Подсветить все клетки, достижимые за N ходов      |
| **«Показать путь»**          | Анимация обхода по всем доступным веткам          |
| **«Открыть окно ходов»**     | Модальное окно со списком ходов (клик — анимация) |
| **«Изменить размер»**        | Перестроить доску с новыми размерами              |
| **«Очистить поле»**          | Сбросить все отметки посещений                    |
| **Спиннер «Глубина»**        | Максимальное количество ходов для BFS             |
| **Спиннеры «Строк/Столбцов»** | Задать новый размер доски                        |

## 🔬 Тестирование

```bash
# Прогон всех тестов
./mvnw test

# Запуск конкретного тестового класса
./mvnw test -Dtest=GameBoardTest
```

Покрытие: `GameBoard` (BFS-алгоритм, стены, граничные условия). Тесты для UI-контроллеров требуют JavaFX runtime и выполняются в отдельном integration-test шаге.
